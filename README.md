# Car-Insurance-Claims-Analysis
# Overview
Using SQL to investigate car insurance claims to identify high-risk groups.
This project analyses demographic, behavioural and vehicle-related attributes of policyholders in order the identify the groups most likely to claim.
The results can be used to adjust premiums to reflect differences in claim behaviour.

Dataset contains 8149 policyholders

# Objectives
Identify high-risk groups within the 12 selected attributes.
